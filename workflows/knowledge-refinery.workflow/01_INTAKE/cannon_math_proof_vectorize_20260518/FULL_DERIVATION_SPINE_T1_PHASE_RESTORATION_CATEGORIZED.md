These corrections are accepted into the living spine. They are not all Lean theorem one. They are lane guards and module queues.

### Correction 1 - T8 Openness Is Posture, Not Operation

Problem:

> If repentance generates `O > 0`, and `O > 0` changes orientation, then a critic can argue T8 is false because a self-generated act changed orientation.

Fix:

> Openness is a posture/property, not the restoring operation. The corrupted system can turn toward the door; it cannot open the door from outside.

Formal lock:

```text
Openness(state) := state oriented toward receiving external input
```

Boundary:

```text
O > 0 and G = 0 -> O*G*(1-C) = 0
```

Posture enables reception. External `G` performs restoration.
