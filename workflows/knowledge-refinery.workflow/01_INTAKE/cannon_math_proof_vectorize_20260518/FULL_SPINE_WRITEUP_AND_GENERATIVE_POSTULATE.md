```text

-> objections / kill conditions
```

The point is not that one register swallows the others.

The point is convergence:

> the same structure appears in human intuition, metaphysics, theology, physics, and formal logic.

## Register 1 — Simple Question

Every section should start with a question that can be understood before technical machinery appears.

Examples:

- Can error be original?
- Can a corrupted system reconstruct its own uncorrupted reference?
- Can the same visible act come from different hidden orientations?
