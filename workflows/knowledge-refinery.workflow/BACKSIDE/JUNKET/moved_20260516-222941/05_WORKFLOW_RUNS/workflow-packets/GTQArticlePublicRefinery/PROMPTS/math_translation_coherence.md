# Math Translation Coherence Prompt

Use when the article contains equations or symbolic notation.

Question:

Is the plain-English math translation coherent?

Return:

- Original equation or notation.
- Word equation.
- Everyday explanation.
- What each symbol means in this article.
- Whether the translation preserves structure.
- What would need to change if it does not.

Failure modes to check:

- symbol meaning drift
- wrong factor order
- metaphor replacing structure
- missing dependency
- spiritual term imposed instead of derived
- claim stronger than equation supports
