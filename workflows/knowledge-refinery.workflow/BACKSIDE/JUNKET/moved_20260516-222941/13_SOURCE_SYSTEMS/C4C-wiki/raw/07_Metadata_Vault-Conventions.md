## Note Structure

### Atomic Notes
- **1 argument/concept per file** (Obsidian best practice)
- **Front matter:** Title, source file, author, date
- **Sections:** Premise → Logic → Conclusion → Objections → References
- **Length:** 500-2000 words typical

### Note Naming
```
[Category]-[Concept]-[Version]

Examples:
- Proof-Logical-Ground-Consilience-v1
- Christian-Designism-Core-Argument
- Divine-Hiddenness-and-Freedom
- Grace-Source-Term-Integration
```
