# Welcome to the Case for Christ Vault

A collaboratively built, rigorously structured examination of the evidence for and against the truth claims of Christianity.

## How This Vault Works

Every major domain of evidence has its own numbered folder. Inside each domain you'll find three subfolders:

- **FOR/** — The strongest arguments supporting Christianity, presented at their best
- **AGAINST/** — The strongest arguments challenging Christianity, presented at their best  
- **KEY-SOURCES/** — Annotated bibliography for that domain

## How to Contribute

1. Fork the GitHub repository
2. Open the vault in Obsidian
3. Pick a domain and write or improve a note using the [[Argument-Template]]
4. Submit a Pull Request to the `review/*` branch
5. Another contributor reviews before merge to `main`

## Vault Rules

1. **Steel-man everything.** Present each argument the way its best advocate would. If you can't make the other side's case compellingly, you don't understand it well enough to refute it.
2. **Cite sources.** Every factual claim needs a citation. Opinions are labeled as opinions.
3. **FOR and AGAINST are equal citizens.** Intellectual honesty above all.
4. **No straw men.** If a note misrepresents a position, tag it `#needs-steelman`.
5. **Tag your confidence.** Use the strength field (1-10) honestly. A brilliant argument with shaky evidence gets a lower score than a modest argument with rock-solid evidence.
6. **Connect, don't silo.** Use `[[wikilinks]]` aggressively. The power of this vault is in the connections between domains.
7. **The Theophysics Bridge (Section 10) is optional** for contributors unfamiliar with the framework.
