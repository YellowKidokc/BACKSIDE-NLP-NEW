## SECTION II — THE CURVE  

When you plot these lifespans against generation number and fit an **exponential decay** model *(exponential decay: a process whose rate of decrease is proportional to its current value)*, you get this:

### Formal Model  

1. Equation (mathematical notation)  
  $L(n)\;=\;L_{0}\,e^{-\,n/\tau}\;+\;L_{\text{floor}}$  

2. Plain-English term-by-term translation  
  $L(n)$ = lifespan at generation *n*  
  $L_{0}$ = initial (pre-decay) lifespan plateau  
  $e^{-\,n/\tau}$ = exponential shrink factor (“lose the same **fraction** each generation”)  
  $\tau$ = decay time-scale (generations required for $L$ to fall to $L_{0}/e$)  
  $L_{\text{floor}$} = lower bound the curve cannot cross  

3. Sensitivity / prediction  
  • Larger $\tau$ → slower decay.  
  • Raising $L_{0}$ shifts the whole curve upward.  
  • Raising $L_{\text{floor}}$ lifts only the asymptote; observed data below the new floor would falsify the model.
