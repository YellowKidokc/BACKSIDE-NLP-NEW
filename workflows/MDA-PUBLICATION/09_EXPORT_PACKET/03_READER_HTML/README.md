# Reader HTML

Open `index.html`.

These pages have four modes:

- Easy
- Standard
- Academic
- Proof

Important: pages without generated Easy/Academic markdown use Standard text as fallback. The top bar tells you whether each page is `ready` or `fallback`.

The Proof tab includes metrics, claim candidates, and the two-lane report. It does not mean the claim has passed final promotion unless the proof gate explicitly says so.
