# Keyword Graph Overlay

This overlay explains the graph edges.

Main files:

- `MDA_KEYWORD_GRAPH_OVERLAY.md`
- `mda_keyword_graph_overlay.json`
- `mda_paper_keywords.csv`
- `mda_bridge_edges.csv`

Use `bridge_terms` to answer: why are these two papers connected?

This is lexical/phrase based. It is not yet a full semantic embedding graph.
