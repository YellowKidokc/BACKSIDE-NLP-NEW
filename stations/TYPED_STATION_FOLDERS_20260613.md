# Typed Station Folders - 2026-06-13

Generic rails stay:

- `INPUT`
- `OUTPUT`
- `EXPORTS`
- `_LOGS`

Reasoning stations should not stop there. They need typed subfolders that describe the work.

## Evidence / Verification Stations

Use these where relevant:

- `INPUT\claims`
- `INPUT\evidence`
- `INPUT\source-packets`
- `OUTPUT\verdicts`
- `OUTPUT\support-map`
- `OUTPUT\refutations`
- `OUTPUT\insufficient-evidence`
- `EXPORTS\evidence-chains`
- `EXPORTS\citations`
- `EXPORTS\audit`
