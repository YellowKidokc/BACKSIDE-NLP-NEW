# Explain It Simple

**The 2-minute version:**  
When you tell a lie, you have to tell more lies to cover it up. This keeps growing until it becomes impossible to keep track. This is like how disorder grows in nature — it’s called entropy. Truth is simple and steady, like a single clear picture. Lies are messy and multiply quickly, like a big tangled web. Over time, lies destroy trust and systems, just like how engines stop working when parts wear out. The Bible’s rules help keep things honest, like filters that catch errors in a message. If you want to last forever, you have to be completely truthful, because even a tiny lie will eventually ruin everything.

**Key terms in plain language:**  
- **Entropy:** A measure of disorder or messiness.  
- **Microstate:** A specific way something can be arranged. Truth has only one way; lies have many.  
- **Noise:** Errors or disturbances that mess up a message.  
- **Signal:** The true message being sent.  
- **Error-correction:** Ways to fix or catch mistakes in messages.  
- **Channel:** The path a message travels through.

**What this is really saying:**  
Sin is like static or noise that messes up the connection between God and people. Lies aren’t just bad choices; they physically break down the system of truth over time. The Bible’s laws and stories are like a guidebook for keeping the message clear and fixing errors. If you keep lying, the system eventually fails — that’s what death means here. Being truthful is the only way to keep the connection strong forever.

**What would prove it wrong, if present:**  
- If lies could exist without needing more lies to support them.  
- If a system with some lies or errors could last forever without collapsing.  
- If truth didn’t correspond to a unique, stable state but could be multiple things.  
- If historical or social data showed no link between honesty and survival of institutions.  
- If biblical teachings didn’t align with any error-correction principles.
