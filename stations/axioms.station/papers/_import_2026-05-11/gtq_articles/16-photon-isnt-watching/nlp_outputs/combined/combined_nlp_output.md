## The Double-Slit Experiment: What Actually Happens

In the lab, electrons are fired one at a time at a barrier with two slits. Behind the barrier is a screen that detects where electrons land.

- Without any detector telling which slit the electron went through, the pattern on the screen shows interference — a wave-like pattern that suggests each electron somehow "knows" about both slits.  
- When a detector is placed to find out which slit the electron passes through, the interference pattern disappears. The electrons behave like particles going through one slit or the other.

The key point: it is not the physical disturbance of measurement that destroys the interference, but the mere possibility of knowing which path the electron took. Information changes the system. This is the core mystery of quantum mechanics.

## PEAR-LAB: Princeton Took This Seriously

PEAR-LAB ran a 28-year experiment with about 2.5 million trials using a Random Event Generator (REG) that produces random bits based on quantum noise.

- When no one tries to influence it, the REG outputs 0s and 1s equally.  
- When a human operator focuses intention to bias the output, the REG’s results shift slightly but consistently in the intended direction.

The effect size per trial is tiny, but the large dataset yields a statistically strong result (about six standard deviations), which in physics is considered a discovery-level signal. Critics have raised valid concerns about replication and methodology, but the effect has not been refuted, only largely ignored.
