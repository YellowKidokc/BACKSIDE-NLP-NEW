# Backside Prompt Packs

**What this is:** Reusable prompts that can be handed to AI partners.
**Owner:** shared
**Status:** live map
**Last updated:** 2026-05-16

Prompt packs are not workflow logs. They are assignment prompts, system prompts, and reusable instructions worth preserving.

Folder names end in `.prompt-pack`:

```text
x-drive-reorg.prompt-pack
paper-citation.prompt-pack
facts-ledger.prompt-pack
timeline-builder.prompt-pack
```

The current prompt bank remains at:

```text
00_WORKFLOWS\prompts\x-drive-reorg\
```

When Backside is fully reorganized, mirror or move it to:

```text
Backside\prompts\x-drive-reorg.prompt-pack\
```

Do not mix live back-and-forth comms here. Use comms workflow rooms for execution conversation and this folder for reusable prompt bodies.
