# Fruits Spirit Canon Station

Builds the canon index for Fruits of the Spirit and their equation mappings.

## Run

```powershell
python Backside\stations\fruits-spirit-canon.station\station.py --out X:\EXPORTS\canon-index\fruits-spirit
```

## Output

- `canon-index.json`
- `canon-index.md`

This station is deterministic. It does not write to Postgres.
