# The Four Falsifiable Predictions

P1: Path 3 agents outperform Path 2 agents on coherence-proxy measures (psychological integration, physiological stress markers, behavioral consistency) controlling for religious activity level.
        P2: Recipients of high-grace experiences are randomly distributed across prior coherence levels — consistent with $G$ being externally sourced rather than earned.
        P3: Conversion and pathway-transition data clusters around threshold events rather than smooth improvement curves.
        P4: Agents in closed attitudinal posture ($s \approx -1$) in grace-rich environments show minimal coherence increase, confirming that $O = 0$ zeroes out $G$.
