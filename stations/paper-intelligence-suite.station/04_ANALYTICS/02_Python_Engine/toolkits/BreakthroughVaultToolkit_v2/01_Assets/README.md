# 01_Assets — Source Material Capture

## Purpose
This is your **intake zone** — where all raw source material enters the system before processing.

## What Goes Here
- **Books** — chapter notes, quotes, page references
- **Videos** — transcripts, timestamps, key moments
- **Articles** — web clippings, PDFs, research papers
- **Podcasts** — episode notes, transcribed sections
- **Scriptures** — verse studies, exegesis notes
- **Conversations** — insights from discussions, mentors, AI

## Folder Structure (Optional)
```
01_Assets/
  ├── Books/
  ├── Videos/
  ├── Articles/
  ├── Podcasts/
  ├── Scripture/
  └── Unsorted/
```

Or keep everything flat and use YAML `source_type:` field to filter.

## Workflow
1. **Capture** — paste raw notes, quotes, links
2. **Use template** — `Ctrl+T` → "01_Asset_Template"
3. **Tag** — add domains, topics, project
4. **Extract** — when ready, pull atomic concepts → `02_Foundations/`

## Status Progression
- `captured` → just logged, unprocessed
- `processing` → actively reading/noting
- `extracted` → atoms/molecules created from it
- `archived` → fully mined, can be stored

## Key Principle
**This is NOT permanent knowledge yet** — it's raw ore waiting to be refined.

Don't overthink organization here. The goal is speed of capture, not perfection.

---

## Quick Start
1. Create new note: `Ctrl+O` → type title → `Shift+Enter`
2. Insert template: `Ctrl+T` → select "01_Asset_Template"
3. Fill in source info
4. Paste raw notes/quotes
5. When done reading, extract key concepts to Foundations
