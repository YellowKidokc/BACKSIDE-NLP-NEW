# 02_Foundations — Atomic Knowledge

## Purpose
This is your **periodic table of ideas** — every note here is a single, context-free, reusable concept.

## What Goes Here
- **Atoms** — irreducible ideas (one concept per note)
- Definitions, principles, observations
- Building blocks for molecules and laws
- Ideas that can stand alone without context

## The Atom Standard
✅ **Good Atom:**
- Contains ONE core idea
- Understandable without reading the source
- Can be reused in multiple contexts
- Clearly titled

❌ **Bad Atom:**
- Mixes multiple ideas
- Requires context from source material
- Too specific to one use case
- Vague or generic

## Examples
**Good:** `Information Cannot Be Destroyed (Conservation)`  
**Bad:** `Stuff from Chapter 3 of that book about quantum things`

**Good:** `Observer Participation Creates Measurement`  
**Bad:** `Quantum Mechanics and Theology Connection`

## Workflow
1. Read source material in `01_Assets/`
2. Identify a single core concept worth isolating
3. Create new note: `Ctrl+O` → title → `Shift+Enter`
4. Insert template: `Ctrl+T` → "02_Foundation_Template"
5. Write in YOUR OWN WORDS (no copy-paste)
6. Link to source material
7. Tag with domains/topics
8. Link to related atoms

## Status Levels
- `baby` → new, unconnected, rough
- `child` → linked, edited, improving
- `adult` → mature, well-connected, referenced in papers

## Trinity Mapping
Every atom should map to a Trinity aspect:
- **Father** — Information, Law, Source
- **Son** — Form, Manifestation, Matter
- **Spirit** — Relationship, Dynamics, Connection

## Key Principle
**Write for your future self** — assume you'll forget this in 6 months.

If you can't understand the note in isolation, rewrite it until you can.
