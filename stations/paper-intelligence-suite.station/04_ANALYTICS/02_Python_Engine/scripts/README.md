# Python Scripts

This directory is intended to hold all the Python scripts for the analytics engine.

Based on the `PYTHON_SCRIPTS_INVENTORY.md` file, the following scripts should be placed here:

## Analysis Scripts
- `analyze_coherence.py`
- `cooccurrence_analyzer.py`
- `extract_metrics.py`
- `matter_analysis.py`

## Chart Scripts
- `chart_templates.py`

## Utilities
- `auto_linker.py`
- `concept_hub_generator.py`
- `duplicate_finder.py`
- `uid_generator.py`
- `validation_scaffold.py`
- `vault_refresh.py`

## Core Scripts
- `api_server.py`
- `grace_vault_manager.py`
- `run_theophysics.py`

These scripts need to be extracted from the markdown files in the `ARCHIVE` directory or recreated based on their descriptions.
