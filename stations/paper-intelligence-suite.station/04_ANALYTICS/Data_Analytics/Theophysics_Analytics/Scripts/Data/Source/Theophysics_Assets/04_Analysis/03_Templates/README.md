# Templates

Note templates organized by vault phase. Use with Templater plugin.

## Template Hierarchy

| # | Template | Phase | Purpose |
|---|----------|-------|---------|
| 01 | Asset_Template | 01_Assets | Raw source material intake |
| 02 | Foundation_Template | 02_Foundations | Atomic concept notes |
| 03 | Analysis_Template | 03_Analysis | Exploration and experimentation |
| 04 | Integration_Template | 04_Integration | Molecule/synthesis notes |
| 05 | Doctrine_Template | 05_Doctrine | Laws and principles |
| 06 | Publication_Template | 06_Publication | Papers and articles |

## Special Templates

- **Master_Equation_Template** - Full YAML schema for master equation notes
- **yaml_frontmatter_template** - Minimal YAML starter template

## Usage

1. Open new note
2. Run: Templater -> Insert Template
3. Select appropriate template for your content phase
4. Fill in the metadata and content sections
