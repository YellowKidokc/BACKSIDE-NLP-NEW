# Scripts

This directory contains all the scripts for the Theophysics project. The scripts are organized by function and language.

## Directory Structure

This directory is not strictly organized by subdirectories, but rather by file naming conventions. Scripts are named to reflect their function. For example, `auto_linker.py` is a Python script that automatically links concepts in the vault.

## Usage

To run a script, navigate to this directory and execute it using the appropriate interpreter. For example, to run a Python script, use:

```bash
python <script_name>.py
```

To run a PowerShell script, use:

```powershell
.\<script_name>.ps1
```
