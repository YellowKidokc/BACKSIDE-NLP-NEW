# Universal AI Vault Builder Prompt

Objective: build a 5-layer vault (Source -> Physics -> Theology -> Information -> Symbol -> Topology).
Each note contains divine_field weights (Theta, Lambda, Psi).

## Process

1. Create a master Source-of-Truth note
2. Generate atoms across layers using templates from `03_Templates/`
3. Run: `python 01_Scripts/run_theophysics.py`
4. Run: `python 01_Scripts/analysis/matter_analysis.py`
5. Iterate on translations and divine links

## Layer Structure

| Layer | Purpose | Content Type |
|-------|---------|--------------|
| Source | Raw biblical/scientific data | Citations, quotes |
| Physics | Scientific frameworks | Equations, theories |
| Theology | Theological frameworks | Doctrines, exegesis |
| Information | Information-theoretic models | Entropy, complexity |
| Symbol | Symbolic representations | Diagrams, metaphors |
| Topology | Relational structures | Networks, hierarchies |
