# System Documentation

This folder contains system-level documentation for the Theophysics Analysis vault.

## Contents

- **Workflow.md** - Instructions for running the analysis pipeline
- **Prompts.md** - AI prompts for vault management
- **Tag_Reference.md** - Complete tag taxonomy for content classification

2. Enable the Obsidian plugin
3. Use Admin Wizard for common tasks
