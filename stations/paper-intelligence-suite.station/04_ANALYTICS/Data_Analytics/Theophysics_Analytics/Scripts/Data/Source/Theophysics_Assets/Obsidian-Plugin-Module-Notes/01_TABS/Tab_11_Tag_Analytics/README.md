# Tab 11: Tag Analytics

**Purpose**: Analyzes tagging patterns and semantic clustering.

**Group**: Progress & Diagnostics  
**Priority**: Phase 4

---

## Overview

This tab provides analytics on tag usage, co-occurrence patterns, evolution over time, and semantic recommendations.

---

## Core Functions

### 1. Tag Frequency
- Count and rank all tags
- Usage trends over time
- Most/least used tags
- Growth rate

### 2. Tag Clusters
- Groups of frequently co-occurring tags
- Semantic relationships
- Cluster visualization
- Pattern detection

### 3. Tag Evolution
- How tagging patterns change over time
- Across papers or periods
- Emerging vs declining tags

### 4. Semantic Recommendations
- AI suggests tags based on content
- Tag-to-ontology mapping
- Synonym detection
- Inconsistency flagging (#causality vs #causal)
