# Wizards

Templater-powered wizards for vault management. Requires:
- Templater plugin (enabled)
- Buttons plugin (optional, for button-based invocation)

## Available Wizards

| Wizard | Purpose |
|--------|---------|
| WIZ - Audit Vault | Generate vault statistics and audit report |
| WIZ - Build MOCs | Create/update Maps of Content |
| WIZ - Build Paper Series | Generate paper series structure |
| WIZ - Link Palette | Generate link snippets for a note |
| WIZ - Setup Folders | Create default folder structure |

## Usage

1. Open a note
2. Run: Templater -> Insert Template
3. Select desired wizard
4. Follow prompts

Or use buttons from `02_System/Admin_Wizard.md`
