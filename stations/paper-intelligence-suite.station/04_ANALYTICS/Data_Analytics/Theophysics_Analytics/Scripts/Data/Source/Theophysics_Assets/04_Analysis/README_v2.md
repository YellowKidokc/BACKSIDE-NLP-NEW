
# Theophysics AI Vault v2 (Living Vault)
This build includes:
- Wizards (Setup, Build MOCs, Paper Series, Link Palette, Audit)
- Python scripts (scan, co-occurrence, auto-linker, validation, hubs)
- Dataview dashboards and hubs
- Ready AI plugin & API (OpenAI/Claude/stub)

## Run
1) Start API: `python scripts/api_server.py`
2) In Obsidian: enable plugin “Theophysics AI Engine” and set endpoint `http://localhost:8001/analyze`.
3) Use Wizards in /Wizards to build MOCs, audit, and link.
4) Open /Dashboards for live views.
