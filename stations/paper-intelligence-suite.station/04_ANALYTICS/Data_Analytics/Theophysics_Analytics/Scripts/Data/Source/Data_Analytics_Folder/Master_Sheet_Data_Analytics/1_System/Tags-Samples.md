## Tag Structure Overview

```
[[pillar/]]          - Foundational domains (physics, theology, etc.)
[[logos/]]           - Conceptual categories (field, grace, decay, etc.)
[[χ_var/]]           - Master Equation variables (G, M, E, S, T, K, R, Q, F, C)
[[math_role/]]       - Mathematical function types
[[paper/]]           - Paper identifiers
[[law/]]             - The 10 Deep Laws
[[miracle/]]         - Jesus miracle categories
[[experiment/]]      - Experimental protocols
```
