# Tab 13: Master Truth Manager

**Purpose**: Controls the master truth folder and its sub-dashboards.

**Group**: Data & Export  
**Priority**: Phase 2

---

## Overview

This tab provides access to the `/master-truth/` folder system, allowing viewing, regeneration, and management of all dashboards.

---

## Core Functions

### 1. Master Sheet Viewer
- Display the unified overview
- Quick navigation to sections
- Search functionality
- Export options

### 2. Dashboard Status
- Health of each sub-dashboard
- Last updated timestamps
- Stale dashboard detection
- Regeneration queue

### 3. Sub-Dashboard Access
- Quick links to all category dashboards:
  - Concepts
  - Math
  - Axioms
  - Evidence
  - Claims
  - Timeline
  - Theories
  - Coherence
  - Breakthroughs
  - External Links
  - Tags
  - Progression

### 4. History Browser
- Navigate timestamped snapshots
- Compare versions
- Restore previous states
- Track changes over time

### 5. Progression Metrics
- Research velocity
- Concept growth rate
- Coherence trend
- Weekly summaries
