# Tab 12: Theory Manager

**Purpose**: Manages your theories (internal and external) as first-class objects.

**Group**: Progress & Diagnostics  
**Priority**: Phase 3

---

## Overview

This tab treats theories as structured entities with their own axioms, claims, evidence, coherence scores, and inter-theory relationships.

---

## Core Functions

### 1. Internal Theory Registry
- Your own frameworks
- Linked axioms, claims, evidence, math
- Per-theory coherence score
- Development status

### 2. External Theory Registry
- Imported frameworks (physics, philosophy, theology)
- Source tracking
- Integration status

### 3. Per-Theory Coherence
- Separate coherence score for each theory
- Component breakdown
- Historical tracking
- Comparison across theories

### 4. Inter-Theory Analysis
- Conflicts between theories
- Dependencies (Theory A relies on Theory B)
- Unification suggestions
- Compatibility matrix

### 5. Dependency Graph
- Visual representation of theory dependencies
- Circular dependency detection
- Foundation identification
