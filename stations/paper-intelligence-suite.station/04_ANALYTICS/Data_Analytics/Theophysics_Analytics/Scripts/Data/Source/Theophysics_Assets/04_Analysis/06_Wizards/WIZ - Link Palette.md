# Link Palette

**Wikilinks**
```markdown
[[path/to/note.md]]
[[path/to/note.md|Display Name]]
```

**Section links**
```markdown
[[path/to/note.md#Introduction]]
[[path/to/note.md#Results|Results Section]]
```

**Block link (after adding a ^block-id)**
```markdown
[[path/to/note.md^key-claim]]
```

**Embeds**
```markdown
![[path/to/note.md]]
![[path/to/note.md#Key Findings]]
```

**Prev / Next / Home (example)**
```markdown
<- [[Previous|Previous]]  -> [[Next|Next]]  [[Home|Home]]
```
