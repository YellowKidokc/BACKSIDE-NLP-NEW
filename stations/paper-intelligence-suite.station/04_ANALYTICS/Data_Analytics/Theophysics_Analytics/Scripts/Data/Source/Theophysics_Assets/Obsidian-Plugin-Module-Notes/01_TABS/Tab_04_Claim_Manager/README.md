# Tab 4: Claim Manager

**Purpose**: Manages assertions and their verification status.

**Group**: Discovery & Analysis  
**Priority**: Phase 2

---

## Overview

This tab displays all claims, their verification status, supporting evidence, and tools for promoting claims to axioms or identifying conflicts.

---

## Core Functions

### 1. Claim Registry
- List all claims with verification status
- Track supporting evidence count
- Show strength assessment
- Filter by verified/unverified

### 2. Verification Tracking
- Mark claims as verified/unverified
- Track evidence requirements
- Calculate verification confidence
- Historical verification tracking

### 3. Claim Promotion
- Identify claims eligible for axiom promotion
- Track promotion history
- Show reasoning for promotions

### 4. Conflict Detection
- Flag conflicting claims
- Show contradiction details
- Suggest resolutions
