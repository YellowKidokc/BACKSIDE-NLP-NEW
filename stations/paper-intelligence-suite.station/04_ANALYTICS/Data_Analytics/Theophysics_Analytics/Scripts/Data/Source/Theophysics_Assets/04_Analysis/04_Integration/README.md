# 04_Integration - Molecules

This folder contains **Molecules** - synthesized concepts formed by combining multiple Atoms.

## What is a Molecule?

A Molecule emerges when two or more Atoms are combined to produce new emergent properties that neither Atom possesses alone.

## Naming Convention

```
MOL_[Primary-Atom]-[Secondary-Atom].md
```

Examples:
- `MOL_Grace-Entropy.md` - The interplay of negentropic and entropic forces
- `MOL_Trinity-Dynamics.md` - How Father-Son-Spirit interact dynamically
- `MOL_Information-Form.md` - How source patterns become manifest structures

Use `03_Templates/04_Integration_Template.md` to create new molecules.
