## 📖 **HOW TO USE THIS FOLDER**

### **Paper Structure**
Each paper folder (P01–P12) contains:
- `000_START_HERE.md` - Quick navigation guide
- `MOST-DEFINITIVE.md` - Current canonical version
- `LATEST-DRAFT.md` - Most recent working draft
- `05_Drafts/` - All draft versions and working notes
- `06_Assets/` - Images, figures, and visual materials
- `08_Publication/` - Publication-ready versions

### **Naming Convention**
- **P##** = Paper number (01-12)
- **Descriptive-Title** = Human-readable paper title
- Example: `P07_The-Grace-Function-A-Negentropic-Cosmological-Model`

## 🎯 **WORKING WITH PAPERS**

### **To Read a Paper:**
1. Navigate to paper folder (e.g., `P01_Logos-Information-Participatory-Universe/`)
2. Open `MOST-DEFINITIVE.md` for current canonical version
3. Or open `LATEST-DRAFT.md` for most recent edits

### **To Edit a Paper:**
1. Create new draft in `05_Drafts/` folder
2. Use descriptive filename (e.g., `Paper-1-Revision-2025-10-13.md`)
3. When finalized, update `LATEST-DRAFT.md` or `MOST-DEFINITIVE.md`
4. Reference [[../Library/Core-Reference/PAPER-EDITING-GUIDE|Editing Guide]]

### **To Add Images:**
1. Place image files in `06_Assets/` folder
2. Use naming convention: `P##_description.png` or `P##B_description.png`
3. Reference in paper with: `![Caption](06_Assets/filename.png)`
4. Update `VISUAL_ASSETS.md` inventory

### **For Publication:**
1. Review [[../Library/Core-Reference/REVIEWER-FIXES-CHECKLIST|Reviewer Checklist]]
2. Apply [[../Library/Core-Reference/PROLEPTIC-DEFENSE-INSERT|Proleptic Defense]]
3. Place final version in `08_Publication/`
4. Draft cover letter using templates
