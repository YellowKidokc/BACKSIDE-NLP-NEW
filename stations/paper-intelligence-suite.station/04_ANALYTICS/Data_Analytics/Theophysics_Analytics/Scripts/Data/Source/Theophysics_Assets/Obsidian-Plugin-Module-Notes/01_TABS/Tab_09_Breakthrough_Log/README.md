# Tab 9: Breakthrough Log

**Purpose**: Tracks the evolution of your understanding and major insights.

**Group**: Progress & Diagnostics  
**Priority**: Phase 4

---

## Overview

This tab records all breakthroughs—both AI-detected and user-marked—showing the progression of your research and major synthesis moments.

---

## Core Functions

### 1. Breakthrough Registry
- Chronological list of all breakthroughs
- Description and significance
- Related items (axioms, evidence, events involved)
- Detection method (AI or user)

### 2. AI-Detected Breakthroughs
- System identifies resolved contradictions
- Detects major unifications
- Finds novel connections
- Calculates significance

### 3. User-Marked Breakthroughs
- Manual tagging of insights
- Custom descriptions
- Importance ratings

### 4. Timeline View
- Visual timeline of breakthroughs
- Clustering analysis
- Velocity tracking (breakthroughs per period)
