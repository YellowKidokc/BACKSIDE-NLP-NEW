# 22_DROP_BOTH_ALIGNMENT

Use this when you want the full mechanism:

1. Paper Intelligence
2. Brain Arm
3. Alignment join

## Option A — Drop files

Put one or more `.md` papers into `INBOX\` and double-click
`RUN_BOTH_ALIGNMENT.bat`.

## Option B — Fetch from somewhere else

Put a full file path or folder path into `FETCH_SOURCE.txt` and double-click
`RUN_BOTH_ALIGNMENT.bat`.

## Output

Each run creates a timestamped folder under `RUNS\`.
