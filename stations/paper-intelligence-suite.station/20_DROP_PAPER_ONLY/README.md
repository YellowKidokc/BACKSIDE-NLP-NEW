# 20_DROP_PAPER_ONLY

Use this when you want to run only the paper-grader side.

## Option A — Drop files

Put one or more `.md` papers into `INBOX\` and double-click `RUN_PAPER_ONLY.bat`.

## Option B — Fetch from somewhere else

Put a full file path or folder path into `FETCH_SOURCE.txt` and double-click
`RUN_PAPER_ONLY.bat`.

## Output

Each run creates a timestamped folder under `RUNS\`.
