# 21_DROP_BRAIN_ONLY

Use this when you want only the vault/brain-side classification and digest.

## Option A — Drop files

Put one or more `.md` papers into `INBOX\` and double-click `RUN_BRAIN_ONLY.bat`.

## Option B — Fetch from somewhere else

Put a full file path or folder path into `FETCH_SOURCE.txt` and double-click
`RUN_BRAIN_ONLY.bat`.

## Output

Each run creates a timestamped folder under `RUNS\`.
